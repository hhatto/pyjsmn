About
-----
pyjsmn is a wrapper for `JSMN`_ (minimalistic JSON parser).
pyjsmn is not support encoding method(obj->str).

.. _`JSMN`: http://zserge.bitbucket.org/jsmn.html


Installation
------------
from pip::

    $ pip install pyjsmn

from easy_install::

    easy_install -ZU pyjsmn


Requirements
------------
Python2.7.


Usage
-----

basic usage::

    >>> import pyjsmn
    >>> pyjsmn.loads('[1, 2, {"test": "hoge"}]')
    >>> [1, 2, {"test": "hoge"}]
